# Portafolio de Mariana Lizette Tovar Salazar

Sitio personal responsive construido con **React + Vite** con estilo inspirado en **shadcn/ui**, paleta rosa, modo oscuro y soporte i18n.

## Características
- Diseño moderno tipo landing con secciones de CV/portfolio.
- Modo claro/oscuro con persistencia en `localStorage`.
- Internacionalización con `es` (default) y `en`.
- Secciones: Hero, Sobre mí, Experiencia, Habilidades, Cursos/Certificaciones, Educación, Idiomas y Contacto.

## Desarrollo
```bash
npm install
npm run dev
```

## Build y lint
```bash
npm run lint
npm run build
```
